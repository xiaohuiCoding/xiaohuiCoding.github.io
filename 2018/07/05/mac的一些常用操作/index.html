mac的一些常用操作

记录一些平时在mac上的常用操作。

1.hosts文件的位置

/etc/hosts

2.CocoaPods仓库的位置

~/.cocoapods/repos

3.系统截图的快捷键组合

shift + command + 3   #自动截取当前窗口中的全部内容，截图存于桌面
shift + command + 4   #可选截取当前窗口中的部分内容，截图存于桌面

4.快速找到已打开的应用程序的快捷键组合

tab + command
