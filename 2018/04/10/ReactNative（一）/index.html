ReactNative（一）

ReactNative简称RN，可以跨平台开发。

这里推荐一个入门的地方

1.搭建开发环境

说白了就是安装一堆必要的软件，比如：Homebrew,Node,Yarn以及Visual Studio Code，还需要一台装有Xcode的Mac。

2.新建项目并运行

react-native init 项目名
cd 项目所在路径
react-native run-ios

3.熟悉项目结构

App.js, app.json, index.js, package.json, yarn.lock五个文件以及node_modules, android, ios三个文件夹

4.写代码前的一些准备

可以先安装一些助力开发的插件，比如：

React Native Tools #基础插件
background #设置代码背景
Document This #连续两次Ctrl+D可快速添加注释模板
Dash #选中后可查询
Auto Rename Tag #自动更新一对标签
Guides #格式化代码，线条对齐 
Path Autocomplete #自动补全文件路径
Project Manager #管理工程
vscode-icons #文件和文件夹图标化

5.配置用户设置（User Settings）

Code／首选项／设置 打开配置文件，也可以用快捷键 command+, 直接打开。

{
    "window.zoomLevel": 1,

    //代码智能提醒（目前新版本的VSCode都自带了这个功能）
    // "files.associations": {
    //     "*.js": "javascriptreact"
    // },

    //修饰符警告忽略
    "javascript.implicitProjectConfig.experimentalDecorators": true,

    "[javascriptreact]": {},

    //格式化代码，线条对齐
    "editor.renderIndentGuides": true,

    "java.errors.incompleteClasspath.severity": "ignore",

    "workbench.iconTheme": "vscode-icons"
}

基本配置如上，也按需增减，相当于插件的作用吧，比如：”editor.renderIndentGuides”: true这一句就相当于起到了安装Guides插件后的作用。
